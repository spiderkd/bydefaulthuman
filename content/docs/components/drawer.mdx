## Installation

```bash
npx shadcn add https://crumble.dev/r/drawer.json
```

## Sides

The rough line is drawn only on the single exposed edge — left, right (default), top, or bottom.

<PreviewContainer
  componentName="drawer"
  code={`<Drawer open={open} onOpenChange={setOpen} side="left">
  <DrawerContent open={open}>...</DrawerContent>
</Drawer>`}
>
  <DrawerSidesDemo />
</PreviewContainer>

## Size

Control width (left/right) or height (top/bottom) with the `size` prop — any CSS value.

```tsx
<Drawer side="right">
  <DrawerContent size="480px">...</DrawerContent>
</Drawer>
```
