# Annotation

Wrap any element with a hand-drawn annotation powered by rough-notation.

---

## Basic

<PreviewContainer
  componentName="annotation"
  code={`import { Annotation } from '@/components/crumble/ui/annotation'

<Annotation type="box">
  <span className="text-base font-medium">Annotated text</span>
</Annotation>`}
>
  <Annotation type="box">
    <span className="text-base font-medium">Annotated text</span>
  </Annotation>
</PreviewContainer>
