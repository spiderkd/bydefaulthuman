## Installation

```bash
npx shadcn add https://crumble.dev/r/card.json
npx shadcn add https://crumble.dev/r/avatar.json
npx shadcn add https://crumble.dev/r/badge.json
npx shadcn add https://crumble.dev/r/toggle.json
npx shadcn add https://crumble.dev/r/separator.json
npx shadcn add https://crumble.dev/r/button.json
```
