## Installation

```bash
npx shadcn add https://crumble.dev/r/stat-card.json
npx shadcn add https://crumble.dev/r/line-chart.json
npx shadcn add https://crumble.dev/r/pie-chart.json
npx shadcn add https://crumble.dev/r/tabs.json
npx shadcn add https://crumble.dev/r/card.json
```
