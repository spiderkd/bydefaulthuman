## Installation

```bash
npx shadcn add https://crumble.dev/r/table.json
npx shadcn add https://crumble.dev/r/input.json
npx shadcn add https://crumble.dev/r/select.json
npx shadcn add https://crumble.dev/r/badge.json
npx shadcn add https://crumble.dev/r/card.json
```
